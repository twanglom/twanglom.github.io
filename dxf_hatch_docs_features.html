Highlights

ไฮไลต์

สร้างลาย Hatch แบบพารามิเตอร์

  • กำหนด pitch, angle, scale, offset พร้อมพรีวิวทันที
  • คลังลาย: บันทึก/โหลดพรีเซ็ต จัดหมวดหมู่ ค้นหา

รูปภาพ → คอนทัวร์ (เวกเตอร์)

  • สไลเดอร์ Threshold และเปิด–ปิดคอนทัวร์รายเส้น
  • ตั้งชื่อเลเยอร์อัตโนมัติ จัดไฟล์ CAD ให้เป็นระเบียบ

Generative AI Stable Diffusion

  • ช่วยออกไอเดียโลโก้ ไอคอน และรูปทรงนามธรรม
  • ปรับคำสั่ง (prompt), สไตล์, seed, ขนาดภาพ
  • ส่งภาพไปที่ Image → Contour เพื่อเวกเตอร์ → DXF ได้ทันที

การวัดและรายงาน

  • วัดระยะ/พื้นที่/เส้นรอบรูป รองรับ polyline/วงกลม/วงรี
  • ส่งออก CSV/TXT สำหรับบันทึกคุณภาพ

การส่งออกเพื่อการผลิต

  • DXF พร้อมใช้งานผลิต โพลีไลน์สะอาด เลือกปิดเส้น (closed)
  • เหมาะกับเลเซอร์มาร์ก/ตัด: ตั้งหน่วยและเลเยอร์ตามขั้นตอนงาน

แบตช์ & กฎเลเยอร์

  • ประมวลผลหลายไฟล์ ปรับสเกลอัตโนมัติ สรุปผลรายไฟล์
  • กำหนดกฎ angle/scale/color ต่อเลเยอร์

AI → Vector → DXF Workflow

  1. Generate — Enter a short prompt (e.g., “minimal monoline falcon logo”). Choose a style preset and size.
  2. Vectorize — Send to Image → Contour; adjust threshold; toggle contours you want.
  3. Export — Output DXF with closed paths and unit scale ready for laser marking/cutting.

AI is optional and designed for ideation. Always validate geometry with Measurement tools before manufacturing.

DXF for Manufacturing

Laser Marking

  • Polyline paths with consistent direction
  • Optional layer per operation (MARK/ENGRAVE/REF)

Laser Cutting

  • Closed paths for cutting; open paths for etch
  • Unit scale control (mm/inch) and safe tolerance

Downstream CAD/CAM

  • Clean entities; avoids duplicate/overlapping paths
  • Target format: DXF R2010 (LWPOLYLINE based)
